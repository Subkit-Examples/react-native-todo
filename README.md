# React Native Todo

> React-Native + Apollo + [SUBKIT.IO](https://subkit.io) GraphQL Todo Example App

![Screencast](assets/screencast.gif)

## iOS

```bash
npm install
react-native link react-native-vector-icons
react-native run-ios
```
